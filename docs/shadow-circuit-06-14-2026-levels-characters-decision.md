# Shadow Circuit 06-14-2026 Levels And Characters Decision

## Decision

Ship four additional authored levels, three more downloaded CC0 soundtrack choices, a first-level mission briefing, and Cinematic-only hero/sentry GLB character assets with the original procedural shapes preserved as the Memory first and Balanced fallback.

## Meshy Character Path

Meshy's docs support the desired workflow:

- Text to 3D creates preview tasks first, with optional refine tasks for texturing.
- Rigging adds a skeleton to standard humanoid/biped models and can provide walking/running animation outputs.
- Animation tasks apply an `action_id` from Meshy's animation library to a rigged character.

The selected target actions are:

- Hero: `30` / `Casual_Walk`
- Sentry: `2` / `Alert`

`MESHY_API_KEY` was not available locally during this checkpoint, so the committed runtime assets are deterministic local fallback GLBs generated by `npm run assets:meshy-characters` with `MESHY_CHARACTER_FALLBACK=true`. The production Meshy script is still present and will replace those files when a real key is available and outputs pass the configured size budget.

## Fallback And Performance

Memory first and Balanced quality keep the original simple runtime player and guard meshes. Cinematic quality preloads:

- `src/assets/characters/hero-cinematic.glb`
- `src/assets/characters/sentry-cinematic.glb`
- `src/assets/objectives/keycard-cinematic.glb`
- `src/assets/objectives/terminal-cinematic.glb`

The character assets are reused on every level and animated in code with small idle/walk/patrol motion so the runtime does not need to keep large skeletal clips loaded. Browser smoke confirmed a 60 FPS sample with Cinematic quality and memory pressure `ok`.

## Level Additions

The new levels are:

- `Service Gallery`: easier perimeter routing with two objectives.
- `Cooling Ducts`: medium difficulty with three objectives and longer patrol lanes.
- `Datacore Maze`: hard route planning through offset walls.
- `Blackout Crown`: final hard level with five sentries and four required objectives.

`npm run test:levels` validates every route point, obstacle clearance, sampled patrol windows, and objective collection before exit unlock.

## Music Additions

The added songs are CC0 MP3 tracks from SRG774's `Dark Sci-Fi Audio Pack` on OpenGameArt:

- `Dark Sci-Fi: Sector`
- `Dark Sci-Fi: Pulse`
- `Dark Sci-Fi: Urgent`

They stay in the same external-download pipeline as the existing soundtrack choices and include metadata in `src/game/audio.ts`.
